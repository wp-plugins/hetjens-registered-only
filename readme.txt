== Description ==

This plug-in restricts the access to blog and feed. Visitors will be forwarded to the login page of wordpress before they can see the blog content. When the access is restricted the feed of the blog will be disabled, but it is possible to activate a private feed for every user of the blog.

== Installation ==

1. Upload the directory hetjens-expiration-date to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings -> Reading, there you find the settings to restrict the access to blog and/or feed.
4. It is activated
